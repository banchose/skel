# nvim notes

## z spelling

- `:set spell?`
- `:set spell spelllang=en_us`

| Key | Action |
|-----|--------|
| `z=` | Show list of suggestions — pick by number |
| `]s` | **Jump** to next misspelling |
| `[s` | **Jump** to previous misspelling |
| `zg` | Add word to your spell file (mark as good) |
| `zw` | Mark word as wrong/bad |
